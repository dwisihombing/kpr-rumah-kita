## 🌟 Tentang Project
**KPR Rumah Kita** adalah aplikasi kalkulator simulasi KPR yang dirancang untuk membantu calon pembeli rumah dalam merencanakan keuangan mereka. Tidak hanya menghitung cicilan, aplikasi ini memberikan gambaran menyeluruh mengenai bagaimana pinjaman Anda bekerja selama masa tenor.

## ✨ Fitur Utama
- **🧮 Simulasi Fleksibel**: Mendukung skema bunga **Fixed (Tetap)** maupun **Tiered (Berjenjang)** yang umum digunakan oleh bank-bank di Indonesia.
- **📅 Jadwal Amortisasi Detail**: Tabel rincian pembayaran pokok dan bunga bulan demi bulan hingga pinjaman lunas.
- **📊 Visualisasi Data**: Grafik proyeksi sisa pinjaman dan komposisi total pembayaran menggunakan **Recharts** untuk memudahkan analisis visual.
- **🌓 Mode Tema & Bahasa**: Mendukung Mode Gelap/Terang serta pilihan Bahasa Indonesia dan Inggris.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, TypeScript.
- **Styling**: Tailwind CSS 4.
- **Charts**: Recharts.
- **Icons**: Lucide React.
- **AI Engine**: Google Generative AI (Gemini API).

## 🚀 Cara Menjalankan Secara Lokal

### Prasyarat
- Node.js (Versi terbaru/LTS)
- API Key dari [Google AI Studio](https://aistudio.google.com/)

### Langkah-langkah
1. **Clone repository ini:**
   ```bash
   git clone https://github.com/dwisihombing/kpr-rumah-kita
   cd KPR-Rumah-Kita
   Instal dependensi:
2. **Instal dependensi:**
   ```bash
   npm install
3. **Konfigurasi Environment:**
   Buat file .env di root folder dan tambahkan API Key Anda:
   ```bash
   GEMINI_API_KEY=your_api_key_here
4. Jalankan aplikasi:
   ```Bash
   npm run dev
   
Buka browser di alamat http://localhost:3000
